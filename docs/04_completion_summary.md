## 🔧 技术实现

### 核心库
- **folium**: 地图展示和标记
- **pandas**: 数据读取和处理
- **JavaScript**: 动画控制

### 关键技术
1. **缓存读取**: 按文件名直接定位，无需扫描
2. **状态分段**: 使用shift和cumsum检测状态变化点
3. **动画控制**: setTimeout + 时间差计算
4. **图层管理**: folium.FeatureGroup + LayerControl

## 📝 使用示例

### 快速查询单车轨迹
```python
from code.map_visualization import plot_vehicle_trajectory

m = plot_vehicle_trajectory(
    vehicle_id=22223,
    start_time='2013-10-22 08:00:00',
    end_time='2013-10-22 10:00:00'
)
m.save('my_map.html')
```

### 查询某时刻车辆分布
```python
from code.map_visualization import plot_minute_snapshot

m = plot_minute_snapshot('2013-10-22 08:00')
m.save('snapshot.html')
```

### 创建动画轨迹
```python
from code.map_visualization import create_animated_trajectory

m = create_animated_trajectory(
    vehicle_id=22223,
    start_time='2013-10-22 08:00:00',
    end_time='2013-10-22 08:30:00',
    speed_factor=200
)
m.save('animation.html')
```

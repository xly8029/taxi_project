# 阶段04 地图基础与轨迹查询 - 使用说明

## 完成功能

✅ **1. 车辆轨迹查询与展示**
- 支持按车辆ID和时间范围查询轨迹
- 自动区分载客（红色）和空载（蓝色）状态
- 标注起点（绿色）和终点（红色）

✅ **2. 按分钟查询车辆位置**
- 从分钟缓存快速读取某时刻所有车辆位置
- 支持筛选指定车辆或限制显示数量
- 载客车辆显示红色，空载车辆显示蓝色

✅ **3. 上下车点分布**
- 显示指定时间段内的上车点（绿色）和下车点（橙色）
- 支持图层切换控制
- 悬停显示详细信息

✅ **4. 单车动画轨迹**
- 车辆图标沿轨迹移动
- 轨迹线逐步绘制
- 根据实际速度和时间差控制动画节奏
- 实时显示车辆状态、速度、时间

✅ **5. 地图选点工具**
- 点击地图获取坐标
- 自动绘制两点间连线
- 为后续路网校正和ETA预测预留接口

## 代码使用示例

### 示例1: 查询特定车辆轨迹

```python
from code.map_visualization import plot_vehicle_trajectory

# 生成车辆22223在指定时间段的轨迹地图
m = plot_vehicle_trajectory(
    vehicle_id=22223,
    start_time='2013-10-22 08:00:00',
    end_time='2013-10-22 10:00:00',
    show_status=True,      # 区分载客/空载状态
    show_markers=True      # 显示起终点标记
)

# 保存地图
m.save('my_trajectory.html')
```

### 示例2: 查询某时刻车辆位置

```python
from code.map_visualization import plot_minute_snapshot

# 查询2013-10-22 08:00时刻的所有车辆位置
m = plot_minute_snapshot(
    time_str='2013-10-22 08:00',
    vehicle_ids=None,      # None表示所有车辆
    max_vehicles=100       # 最多显示100辆车
)

m.save('snapshot_0800.html')
```

### 示例3: 查询上下车点分布

```python

    start_time='2013-10-22 08:00:00',
    end_time='2013-10-22 09:00:00',

```

### 示例4: 创建动画轨迹


## 数据源说明

### 车辆缓存
- 路径: `data/cache/vehicle/{车辆id}.csv`
- 用途: 快速查询单个车辆的完整轨迹
- 文件数: 11,120个（每辆车一个文件）

### 分钟缓存
- 路径: `data/cache/minute/2013-10-22/{HH-MM}.csv`
- 用途: 快速查询某一分钟所有车辆的位置
- 文件数: 1,440个（一天24小时×60分钟）

### OD缓存
- 路径: `data/cache/od/od_cache.csv`
- 用途: 订单起终点信息（已剔除异常订单）
- 订单数: 483,168条

## 性能优化建议

1. **轨迹查询**: 
   - 时间跨度不要太大（建议≤2小时）
   - 数据点过多时会导致地图渲染缓慢

2. **位置快照**:
   - 控制max_vehicles参数（建议≤500）
   - 车辆过多会导致浏览器卡顿

3. **上下车点**:
   - 控制max_points参数（建议≤300）
   - 时间跨度不要太大

4. **动画轨迹**:
   - 时间跨度建议≤30分钟
   - 调整speed_factor控制播放速度
   - 数据点过多时可先抽样

## 技术细节

### 坐标系统
- 使用WGS84坐标系（经纬度）
- 深圳市中心: [22.52847, 114.05454]
- 坐标格式: [纬度, 经度]

### 颜色方案
- 载客状态: #FF4444 (红色)
- 空载状态: #4444FF (蓝色)
- 上车点: #00CC00 (绿色)
- 下车点: #FF8800 (橙色)

### JavaScript动画原理
- 使用setTimeout实现逐帧动画
- 根据相邻GPS点的时间差和distance计算延迟
- 通过speed_factor参数加速播放
- 轨迹线使用L.polyline动态添加点

## 常见问题

### Q: 为什么地图加载很慢？
A: 数据点过多。建议缩短时间范围或减少显示数量。

### Q: 动画播放太快/太慢？
A: 调整speed_factor参数，数值越大播放越快。

### Q: 如何查看更多车辆？
A: 修改函数调用中的vehicle_id参数，可用车辆ID见data/cache/vehicle/目录。

### Q: 地图选点的坐标在哪里？
A: 在弹窗中显示，同时在浏览器控制台（F12）中也会输出。
